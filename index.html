زيارة الموقع

تلاوة القرآن الكريم مع خيارات عرض وترجمة متعددة، بتصميم يشبه المصحف الورقي.

زيارة الموقع

تطبيق قرآن شامل يسمح بقراءة القرآن الكريم، الاستماع إليه حسب الجزء، السورة أو الآية، وقراءة الترجمات المختلفة جنباً إلى جنب مع النص العربي. يمكنكم بناء تطبيقاتكم باستخدام API المفتوح الخاص به والاستفادة من ملفات الصوت عالية السرعة عبر CDN.
